# MeuFinanceiro

Aplicativo de controle financeiro pessoal desenvolvido na disciplina de Programação Para Dispositivos Móveis II (IFPE – Campus Garanhuns).

## Objetivo do App

Ajudar o usuário a registrar suas receitas e despesas de forma simples, organizar categorias de gastos e visualizar um resumo financeiro (saldo, total de receitas e despesas) diretamente pelo celular.

## Funcionalidades (MVP)

- Registro de receitas e despesas
- Listagem de transações
- Cálculo de saldo, total de receitas e total de despesas
- Organização por categorias
- Armazenamento local dos dados

## Tecnologias

- Android Studio
- Linguagem: Kotlin
- Armazenamento local: (SharedPreferences ou Room – a definir)
